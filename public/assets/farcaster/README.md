# Farcaster Mini App Assets

This directory contains the required assets for the Lucidly Farcaster Mini App.

## Required Assets

### 1. Screenshots (1284x2778px portrait)
- `screenshot1.png` - Main yield opportunities view
- `screenshot2.png` - Portfolio/dashboard view  
- `screenshot3.png` - Detailed yield strategy view

### 2. Hero Image (1200x630px)
- `hero.png` - Promotional image for app store (1.91:1 aspect ratio)

### 3. Open Graph Image (1200x630px)
- `og-image.png` - Image shown when shared on social media (1.91:1 aspect ratio)

## Image Requirements

### Screenshots
- **Dimensions**: 1284x2778px (portrait)
- **Format**: PNG
- **Content**: Should showcase key features of your yield platform
- **Quality**: High resolution, clear and professional

### Hero & OG Images
- **Dimensions**: 1200x630px
- **Format**: PNG
- **Aspect Ratio**: 1.91:1
- **Content**: Should include Lucidly branding and value proposition

## Creation Guidelines

1. **Screenshots**: Take actual screenshots of your app in action
2. **Hero Image**: Create a promotional image highlighting your platform's benefits
3. **OG Image**: Design for social media sharing with clear branding
