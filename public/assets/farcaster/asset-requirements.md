# Asset Creation Guide for Lucidly Farcaster Mini App

## Required Assets to Create

### 1. Screenshots (1284x2778px portrait)

**screenshot1.png** - Main Yield Opportunities View
- Show your main yield opportunities page
- Display different yield strategies (Base, Ethereum, Stablecoins)
- Include APY percentages and key metrics
- Use mobile viewport for proper aspect ratio

**screenshot2.png** - Portfolio/Dashboard View
- Show user portfolio overview
- Display total value locked, earnings, etc.
- Include charts or graphs showing performance
- Highlight the dashboard functionality

**screenshot3.png** - Detailed Yield Strategy View
- Show a specific yield strategy in detail
- Include deposit/withdraw interface
- Display risk metrics and historical performance
- Show the detailed analysis view

### 2. Hero Image (1200x630px)

**hero.png** - Promotional Image
- Include "Lucidly" branding prominently
- Add tagline: "Advanced Yield Platform"
- Show visual elements representing DeFi/yield farming
- Use your brand colors (blue gradient background)
- Include call-to-action or value proposition

### 3. Open Graph Image (1200x630px)

**og-image.png** - Social Media Image
- Similar to hero image but optimized for social sharing
- Include "Lucidly" logo and name
- Add subtitle: "Maximize Your DeFi Returns"
- Use high contrast for better visibility on social platforms
- Include visual elements that represent yield farming

## Design Guidelines

### Colors
- Primary: Blue gradient (#1e3a8a to #3b82f6)
- Secondary: White text for contrast
- Accent: Gold/yellow for yield/earnings

### Typography
- Use your existing brand fonts
- Ensure text is readable at small sizes
- Keep text minimal and impactful

### Content Focus
- Emphasize "yield" and "earn" concepts
- Show DeFi/blockchain visual elements
- Highlight the multi-chain aspect
- Focus on user benefits and returns

## Tools for Creation

### Screenshots
1. Use browser dev tools to set viewport to 1284x2778
2. Take screenshots of your actual app
3. Use tools like Figma or Canva for editing

### Hero/OG Images
1. Use Figma, Canva, or Adobe Creative Suite
2. Start with 1200x630px canvas
3. Export as PNG with high quality

## File Naming Convention
- screenshot1.png
- screenshot2.png  
- screenshot3.png
- hero.png
- og-image.png
